# Ligthos

Lightos is a Node.js template for building small microservice architectures fast. I provide you only the source, and so that you can flexibly customize it with your own preferences.

## Installation

```bash
git clone ...
```

## Usage

```bash
cd lightos/src/
npm run dev
```
